# quantmetrics-analytics

Read-only analytics on QuantLog JSONL. See `docs/ANALYTICS_ARCHITECTURE.md` and `docs/ANALYTICS_SPRINT_PLAN.md`. **VPS sync + lokale backtest:** `docs/LIVE_VPS_AND_LOCAL_BACKTEST.md`.

**Run:** `python -m quantmetrics_analytics.cli.run_analysis --jsonl <file.jsonl>`

Install: `pip install -e .` from this directory.
